## Context

Different stakeholders required different reporting formats. Task-level detail for the direct manager, KPI-level summary for the CPO, project timelines for Business, and ticket-level tracking for the TPM. PMs were spending significant time reformatting the same information for multiple audiences.

## Approach

- Created a single source-of-truth document covering all relevant dimensions of project status.
- Used AI to generate four distinct format versions from the same source document, calibrated for each stakeholder audience.

## Result

- Significantly reduced PM time spent on stakeholder reporting.
- Improved consistency and accuracy across all reporting formats.
- Demonstrated a reusable pattern for AI-assisted PM workflows.
