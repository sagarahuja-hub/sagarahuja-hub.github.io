# sagarahuja.co

Personal portfolio site. Built with Astro + Markdown content collections.

## Run locally

```
npm install
npm run dev
```

Site runs on `http://localhost:4321`.

## Add a new project

1. Create a markdown file in `src/content/projects/`.
2. Fill in frontmatter (see existing files for schema).
3. Set `featured: true` if it should appear on the homepage.

## Deploy

Site is a static build. Run `npm run build`, then deploy the `dist/` folder to Vercel, Netlify, Cloudflare Pages, or GitHub Pages.

Custom domain: `sagarahuja.co`.
